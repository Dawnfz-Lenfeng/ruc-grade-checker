# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 进行版本发布。

## [0.2.0] - 2024-04-01

### 新增
- 使用 typer 重写命令行界面
- 添加了表格形式的成绩展示

### 改进
- 使用 pyproject.toml 替代 requirements.txt
- 优化了项目结构，更符合 Python 包标准
- 改进了错误处理和提示信息

## [0.1.0] - 2024-01-21

### 新增
- 首次发布
- 支持成绩查询和导出
- 支持导出 PDF 成绩单
- 支持保存为 Excel 和 CSV 格式
